# GAEST305-DIGITAL-ELECTRONICS-AND-LOGIC-DESIGN

1.	Course Overview:

The Logic Circuit Design course presents the knowledge and skills needed to design and implement digital circuits. Electronic circuits can be broadly classified into analog and digital circuits. Digital circuits comprise both sequential and combinational circuits i.e., systems with memory and without. Digital circuits deal with binary logic i.e., 1s and 0s. Binary logic and systems that follow this number system have operations, laws, and applications that are entirely different from conventional systems. 

This course offers a comprehensive introduction to digital logic principles, equipping students with both theoretical foundations and practical design skills essential for computer system architecture 
and digital hardware development. It blends classical circuit design techniques with modern hardware description languages like Verilog, emphasizing both combinational and sequential logic design.

2.	Course Objectives:

1.	To familiarize the basic concepts of Boolean algebra and digital systems.
2.	To enable the learner to design simple combinational and sequential logic circuits, which is essential in understanding the organization & design of computer systems.
